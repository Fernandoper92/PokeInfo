SOBRE O POKEINFO



    O PokeInfo é uma plataforma que funciona como um catálogo de pokemons,desenvolvida para meu treinamento com Angular e TypeScript, meu objetivo com ela é colocar o maximo de funcionalidades diferentes para que eu possa praticar. Escolhi essa API de pokemon para meu primeiro projeto por ser uma API um pouco complicada e com isso exigindo mais de mim.

    No catálago você consegue ver atraves de cards todos os pokemons, podendo fazer uma buscar por nome ou filtrar por geração. passando o mouse por cima de cada card é possivel ver informações adicionais e se clicar no card é aberto um overlay aonde mostra o restante das informações, também é possivel salvar o pokemon como favorito clicando na estrela, lembrando que os seus favoritos são salvos no localStorage do navegador.

    Atualmente é possivel ver informações como tipo, peso, altura, id, evoluções, descrição( sómente em inglês e com alguns bugs). Só permiti o catálogo até a terceiro geração de pokemons porque o site de onde pego as imagens está com problema nas gerações seguintes.


    O PokeInfo utiliza a API https://pokeapi.co/ para dados e o site https://pokeres.bastionbot.org para as imagens.
